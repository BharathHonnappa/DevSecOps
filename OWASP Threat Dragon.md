# OWASP Threat Dragon

https://owasp.org/www-project-threat-dragon/

It is a tool that is used to create threat models and identify threats in the system.

Supports creating data flow diagrams, which are very helpful for identifying system flaws.

It includes a rule engine that automatically suggests threats based on components, using frameworks like STRIDE, LINDDUN, and CIA.

We can run it as a Desktop app for local files or deploy it via Docker as a web app that syncs directly with GitHub/GitLab.

Files can be saved in JSON format, which helps version-control the threat models right next to the source code

Download link - https://github.com/OWASP/threat-dragon/releases/tag/v2.6.0


For more information, we can refer 

https://www.threatdragon.com/docs/
